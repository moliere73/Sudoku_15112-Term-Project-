 A Classic Sudoku Game
(featuring some phone and game elements)
There are different levels to choose from, different hints to use, and many other colorful features

To run this game offline:
1. Download CMU CS3 graphics https://academy.cs.cmu.edu/desktop
2. Download PIL (because there are a lot of images) https://pillow.readthedocs.io/en/stable/installation.html
3. Unzip all files in tp3
4. Run main.py in the Python editor 
5. Play the game
